# CheckSum-Arduino (Low SRAM Version)

# 專案架構
 ```
  |-- CheckSum,
      |-- README.md,
      |-- include,
      |-- lib,
      |   |-- BoolArray,
      |   |-- CheckSum,
      |   |-- LinkedList,
      |-- src,
      |   |-- main.cpp,
      |-- test,
          |-- README,
          |-- test.py,
 ```
 
 # 如何使用
**`安裝 Lib `**

- 把專案目錄裡lib資料夾的東西全部移到Arduino libraries資料夾裡面

**`更改副檔名`**

- 把專案目錄裡src資料夾裡面的main.cpp的副檔名改成 .ino

**`開啟當按`**

- 用Arduino IDE點開 main.ino
